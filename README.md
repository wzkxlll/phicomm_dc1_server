
# 斐讯DC1插板服务器端

## 服务架构说明
### 🔧 1. 设备通信服务（`device_server_proj`）
- **强制端口**：`8000`（不可更改，硬件设备依赖此端口）
- **技术框架**：Python脚本
- **核心功能**：与DC1通信
- **目录文件**：`/device_server_proj/README.md`（部署指南）

### 🖥 2. 网页控制台（`web_console_proj`）
- **监听端口**：可自定义（默认使用`8001`，按需修改）
- **技术框架**：Django Web服务器
- **目录文件**：`/web_console_proj/README.md`（配置说明）
- **核心功能**：控制设备的网页端
